# Cours de Programmation Avancée

Ce répertoire contient une série de notebooks Jupyter qui présentent les concepts de programmation avancée.

## Prérequis

- Un terminal en ligne de commande (Mac/Linux ou Windows).
- Python 3.13.11 recommandé.

### Installation de l'environnement virtuel Python

- **Mac/Linux**

  ```bash
  python3 -m venv .venv
  source .venv/bin/activate
  python -m pip install --upgrade pip
  python -m pip install -r requirements.txt
  ```

- **Windows**

  ```powershell
  python -m venv .venv
  .venv\Scripts\activate
  python -m pip install --upgrade pip
  python -m pip install -r requirements.txt
  ```

**Notes :**
- `requirements.txt` inclut les dépendances en Python des notebooks.
